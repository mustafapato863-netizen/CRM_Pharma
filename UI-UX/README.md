# UI-UX

This folder contains the new frontend presentation layer for the CRM system.

## Goals
- Deliver a modern visual system without changing backend or database behavior.
- Keep business logic and data flow intact.
- Provide reusable UI primitives for future screen redesigns.

## Structure
- components/ — reusable UI components
- layouts/ — page shell and navigation patterns
- styles/ — tokens and shared class definitions
- assets/ — static visual assets
