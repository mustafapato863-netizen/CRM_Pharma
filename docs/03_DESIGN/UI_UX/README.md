# UI UX Reference

This folder is the source of truth for the redesign direction of Pharmacy CRM.

Use these files when rebuilding screens:
- `00-system-principles.md` for visual language and product rules.
- `01-shell-and-layout.md` for shared page structure and states.
- `02-workspace-patterns.md` for per-page behavior and hierarchy.
- `03-page-map.md` for the page-by-page reference.

The goal is not to invent new UI per page.
The goal is to make every screen feel like the same product, with each workspace optimized for its primary job.
